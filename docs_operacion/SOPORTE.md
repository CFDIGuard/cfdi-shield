# Alcance de Soporte - CFDI Shield

## 1. Que incluye el soporte

El soporte de CFDI Shield puede incluir:

- instalacion inicial
- configuracion de `.env`
- apoyo para arranque con `install.bat` y `start.bat`
- apoyo para configurar NSSM
- revision de errores operativos controlados
- diagnostico de carga XML
- diagnostico de exportacion Excel
- orientacion sobre modo local y validacion SAT

## 2. Tiempos sugeridos de atencion

Estos tiempos son una recomendacion operativa y pueden ajustarse comercialmente:

- incidencias criticas: 4 a 8 horas habiles
- incidencias medias: 1 dia habil
- solicitudes de ajuste menor: 2 a 5 dias habiles
- dudas de uso: mismo dia o siguiente dia habil

## 3. Que no incluye

Salvo pacto expreso, el soporte no incluye:

- asesoria fiscal formal
- dictamen contable
- interpretacion legal
- desarrollo de funcionalidades nuevas no contratadas
- soporte a infraestructura ajena al sistema cuando no haya acceso o control
- configuracion de red corporativa, firewall o antivirus del cliente
- recuperacion de datos no respaldados

## 4. Requisitos para recibir soporte

Para una atencion mas rapida se recomienda compartir:

- descripcion clara del problema
- hora aproximada del incidente
- capturas de pantalla
- mensaje de error visible
- archivo `app-debug.log` si aplica
- confirmacion de si el problema ocurre en modo local o con SAT habilitado

## 5. Casos frecuentes de soporte

- no inicia la aplicacion
- no valida contra SAT
- no llegan correos
- no carga XML
- marca duplicados
- no exporta Excel
- no abre dashboard
- fallo en login, reset o 2FA

## 6. Contacto

El canal de soporte oficial debe definirse por el responsable comercial de CFDI Shield.

Ejemplo:

- correo: `soporte@cfdi-shield.local`
- horario: `Lunes a Viernes, 9:00 a 18:00`
