# Demo Comercial - CFDI Shield

## Nombre comercial provisional

- Marca visible actual: `CFDI Shield`
- Servicio Windows: `CFDIShield`
- Instalador esperado: `CFDIShieldSetup.exe`

## Objetivo de la demo

Presentar CFDI Shield como una solucion local para:

- cargar XML CFDI
- detectar duplicados
- validar contra SAT cuando esta habilitado
- revisar riesgos y proveedores
- exportar reportes a Excel

## Mensaje comercial corto

CFDI Shield te ayuda a controlar CFDI en un entorno local, sin subir informacion a la nube y con una operacion pensada para auditoria, seguimiento y control interno.

## Alcance de RR9

El analisis RR9 de CFDI Shield evalua riesgo por proveedor con base en CFDI cargados: concentracion, volumen, cancelaciones, repeticion, multimoneda e informacion incompleta. No analiza accionistas, consejeros ni estructura corporativa.

## Flujo recomendado

1. Login
2. Carga XML
3. Dashboard
4. Exportacion Excel
5. Toggle SAT

## Nota de marca

Este nombre se usa como marca visible y comercial antes del empaquetado final del instalador. Los modulos internos de Python pueden conservar nombres tecnicos previos mientras no afecten la estabilidad del sistema.
